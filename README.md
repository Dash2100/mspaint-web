# MS Paint (Win10) Web Clone

This is a browser-based clone of the classic Windows 10 MS Paint (Paint — Web Edition v1.0). The entire application is contained within a single HTML file, requiring no installation, external dependencies, or backend servers. Just open it in your browser and start drawing!

## Features

* **Zero Installation**: Everything is packaged into a single `.html` file. Simply open it in any modern web browser to use it.
* **No Frameworks**: Built entirely with native HTML5 Canvas, CSS3, and Vanilla JavaScript (ES6).
* **Dark Mode Support**: Built-in toggle for light and dark themes to provide a more comfortable viewing experience.
* **Classic UI**: Faithfully recreates the Windows 10 Paint interface, including the Ribbon tabs (Home / View), Quick Access Toolbar, and Status bar.

## Core Functions

* **Basic Drawing Tools**: Includes Pencil, Fill with color (Paint bucket), Text, Eraser, Color picker, and Magnifier.
* **Rich Brushes**: Supports standard Brush, Calligraphy brushes, Airbrush, Oil brush, Crayon, Marker, Natural pencil, and Watercolor brush.
* **Geometric Shapes**: Built-in shapes including lines, curves, rectangles, polygons, stars, arrows, callouts, and hearts (over 20 in total). Supports various outline and fill styles (e.g., solid, crayon, watercolor).
* **Image & Selection Operations**: Supports rectangular selection, transparent selection, moving, cropping, resizing/skewing, and multiple angles for rotating and flipping.
* **Color Management**: Features the classic 10x2 default color palette and a custom "Edit Colors" dialog supporting RGB and HSL adjustments.
* **File I/O**: Open local images (supports drag-and-drop and clipboard pasting), and save your artwork as PNG, JPEG, BMP, GIF, or WebP.
* **History**: Supports up to 50 levels of Undo and Redo operations.

## Keyboard Shortcuts

* **Ctrl + S**: Save file / **Ctrl + Shift + S**: Save as
* **Ctrl + Z**: Undo / **Ctrl + Y**: Redo
* **Ctrl + C**: Copy / **Ctrl + X**: Cut / **Ctrl + V**: Paste
* **Ctrl + A**: Select all
* **Ctrl + + / -**: Zoom in / Zoom out
* **Ctrl + Mouse Wheel**: Zoom canvas
* **F11**: Toggle full screen
* **Delete**: Delete selection

## How to Use

1. Download the `paint.html` file to your computer.
2. Double-click the file or open it using any modern web browser (e.g., Google Chrome, Microsoft Edge, Mozilla Firefox, or Safari).
3. Start creating!

## Requirements

* A modern web browser with HTML5 Canvas and ES6 JavaScript support.
* Mouse, trackpad, or basic touch/stylus pointer input.
